## Design Centrado na Comunicação – Cenário 5: Dashboard Inicial, Histórico e Timeline (Victor Caldeira Iak)

| tópico > subtópico (diálogo) | falas e signos |
|-----------------------------|----------------|
| acesso ao histórico de processamentos | U: Preciso visualizar meus **processamentos** anteriores. |
| > acessar dashboard inicial | D: O sistema apresenta o **histórico** de atividades realizadas. |
| > visualizar histórico | U: Quero identificar um **processamento** específico. |
|  | D: O sistema exibe os registros com **informações** relevantes. |
| > reconhecer processamento | U: Quero confirmar se encontrei o **item** correto. |
|  | D: O sistema apresenta **dados** que permitem identificar cada **processamento**. |
| > selecionar processamento | U: Quero reabrir um **resultado** anterior. |
|  | D: O sistema carrega os **dados** do **processamento** selecionado. |
| > retomar análise | U: Quero continuar o trabalho a partir desse **resultado**. |
|  | D: O sistema permite acessar e reutilizar os **dados** carregados. |
| > iniciar novo fluxo | U: Quero iniciar um novo **processamento**. |
|  | D: O sistema permite começar um novo **fluxo** de trabalho. |
| > navegar entre atividades | U: Quero alternar entre diferentes **processamentos**. |
|  | D: O sistema mantém o **histórico** acessível e organizado. |
