# Avaliação Heurística de Usabilidade

**Nome do Avaliador (especialista):** Vinicius Saide

**Nome do Avaliado (testado):** GeoSegment

---

## Legenda

| Valor | Significado |
|-------|-------------|
| -1 | Não atende as recomendações |
|  0 | Atende parcialmente as recomendações |
|  1 | Atende completamente as recomendações |

---

## Resumo dos Resultados

| Resultado | Quantidade |
|-----------|-----------|
| Heurísticas que **não** atendem as recomendações | 4 |
| Heurísticas que atendem **parcialmente** as recomendações | 12 |
| Heurísticas que atendem **completamente** as recomendações | 11 |

---

## Heurísticas

### 1. Cores

| # | Heurística | Nota (-1 / 0 / 1) |
|---|-----------|:-----------------:|
| 1.1 | O site deve utilizar (aproximadamente) cores padrões de links. | 1 |
| 1.2 | Os botões devem estar em um formato e tamanho padrão. | 0 |
| 1.3 | O plano de fundo da página não pode ser propenso a distrair o usuário do foco do site. | 1 |

---

### 2. Conteúdo

| # | Heurística | Nota (-1 / 0 / 1) |
|---|-----------|:-----------------:|
| 2.1 | Todas as fontes devem ser as mesmas. | 1 |
| 2.2 | Os textos devem estar apropriadamente alinhados. | 1 |
| 2.3 | Logo e textos de cabeçalho devem estar alinhados à esquerda. | 1 |
| 2.4 | Evitar o uso de caixa-alta em textos. | 1 |

---

### 3. Imagens

| # | Heurística | Nota (-1 / 0 / 1) |
|---|-----------|:-----------------:|
| 3.1 | Todas as imagens devem estar alinhadas apropriadamente. | 1 |
| 3.2 | Deve-se garantir que os botões que representam comandos estejam todos com tamanhos e formas similares, além de fonte e tamanho de fonte consistentes. | 0 |
| 3.3 | O formato dos banners (estilo, tamanhos e forma de exibição) devem ser os mesmos em todas as seções do site. | 1 |
| 3.4 | O texto deve se ajustar corretamente ao redor das imagens e possuir um espaçamento, para que não passe a impressão de estar "aglutinado" à imagem. | 1 |
| 3.5 | As páginas devem ser visualmente consistentes mesmo sem imagens. | 0 |

---

### 4. Instruções

| # | Heurística | Nota (-1 / 0 / 1) |
|---|-----------|:-----------------:|
| 4.1 | As mensagens de erro e os textos de ajuda nos campos de formulários devem estar grafadas corretamente. | 1 |
| 4.2 | Todos os campos e botões habilitados devem possuir texto de ajuda. | 0 |
| 4.3 | Mensagens de progresso devem estar presentes em ações que são divididas entre etapas durante várias páginas. | -1 |

---

### 5. Navegação

| # | Heurística | Nota (-1 / 0 / 1) |
|---|-----------|:-----------------:|
| 5.1 | Todos os campos desabilitados e somente leitura devem ser evitados na navegação entre os campos do formulário pela tecla TAB. | 1 |
| 5.2 | As barras de rolagem devem aparecer se necessário. | 1 |
| 5.3 | Deve haver um link para a página inicial em cada página. | 1 |

---

### 6. Usabilidade

| # | Heurística | Nota (-1 / 0 / 1) |
|---|-----------|:-----------------:|
| 6.1 | Todos os rótulos dos campos de formulário devem ser escritos corretamente e de maneira clara. | 1 |
| 6.2 | As fontes não devem ser muito grandes nem tampouco muito pequenas para ler. | 1 |
| 6.3 | Os nomes em botões de comando e caixas de opção não devem estar abreviados. | 1 |
| 6.4 | Garantir que caixas de opções, botões de opção e botões de comando estão logicamente agrupados em áreas de agrupamento claramente demarcadas. | 1 |
| 6.5 | As páginas devem ser impressas de modo legível sem cortar o texto. | 0 |
| 6.6 | O site deve possuir um visual consistente e claramente reconhecível. | 1 |
| 6.7 | Caso o site possua uma área de login, os usuários devem possuir a opção de efetuar login utilizando tanto um nome de usuário quanto um e-mail como ID. | -1 |
| 6.8 | O site deve se ajustar às resoluções de tela mais comuns. | 1 |
| 6.9 | A terminologia do site deve ser adequada aos usuários-alvo do site. | 0 |
