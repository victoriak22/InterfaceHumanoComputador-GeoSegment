## Tabela de Problemas e Recomendações

| Objetivos / Operações | Problemas e Recomendações |
|---|---|
| **0. Comparar resultados de períodos diferentes (1 → 2 → 3 → 4)** | **Input:** dois mapas segmentados de períodos diferentes <br> **Feedback:** visualização de mapa comparativo, gráficos e estatísticas de variação <br> **Plano:** selecionar períodos → ativar comparação temporal → analisar mudanças → utilizar resultados na tomada de decisão <br> **Recomendação:** fornecer visualização clara das mudanças entre períodos |
| **1. Selecionar imagens processadas (1.1 → 1.2 → 1.3)** | **Plano:** abrir dashboard → escolher período atual → escolher período anterior <br> **Problema:** usuário pode ter dificuldade em localizar resultados antigos <br> **Recomendação:** exibir lista organizada por data ou área analisada |
| **1.1 Abrir Dashboard** | **Problema:** usuário pode não saber onde acessar a funcionalidade de comparação <br> **Recomendação:** destacar opção de comparação temporal no menu principal |
| **1.2 Escolher período atual** | **Problema:** usuário pode selecionar imagem incorreta <br> **Recomendação:** exibir miniatura e informações da imagem (data, local) |
| **1.3 Escolher período anterior** | **Problema:** dificuldade em identificar o período correto para comparação <br> **Recomendação:** permitir ordenação por data e filtros de busca |
| **2. Ativar comparação temporal (2.1 → 2.2)** | **Plano:** clicar em comparação temporal → sistema sobrepõe mapas <br> **Problema:** usuário pode não compreender como funciona a sobreposição <br> **Recomendação:** mostrar legenda ou explicação visual da comparação |
| **2.1 Clicar em Comparação temporal** | **Problema:** botão pode não ser facilmente identificado <br> **Recomendação:** utilizar ícone e texto descritivo claro |
| **2.2 Sistema sobrepõe mapas** | **Problema:** sobreposição pode dificultar visualização de detalhes <br> **Recomendação:** permitir ajuste de transparência ou alternância entre mapas |
| **3. Analisar mudanças (3.1 → 3.2 → 3.3)** | **Plano:** identificar áreas alteradas → verificar variação percentual → visualizar gráficos comparativos <br> **Problema:** usuários podem ter dificuldade em interpretar mudanças complexas <br> **Recomendação:** destacar áreas alteradas com cores contrastantes |
| **3.1 Identificar áreas de alteração** | **Problema:** mudanças pequenas podem passar despercebidas <br> **Recomendação:** utilizar cores fortes ou marcadores visuais |
| **3.2 Ver variação percentual por classe** | **Problema:** valores percentuais podem ser difíceis de interpretar isoladamente <br> **Recomendação:** apresentar comparação lado a lado |
| **3.3 Visualizar gráficos comparativos** | **Problema:** excesso de classes pode dificultar leitura do gráfico <br> **Recomendação:** destacar classes com maior variação |
| **4. Utilizar resultado (4.1 → 4.2 → 4.3)** | **Plano:** interpretar impacto ambiental → incluir dados em relatório → compartilhar resultados <br> **Problema:** usuário pode não saber como exportar ou compartilhar análise <br> **Recomendação:** incluir opção clara de exportação ou compartilhamento |
| **4.1 Interpretar impacto ambiental** | **Problema:** interpretação pode exigir conhecimento técnico <br> **Recomendação:** apresentar resumo interpretativo automático |
| **4.2 Incluir dados em relatório** | **Problema:** integração com relatórios pode ser complexa <br> **Recomendação:** gerar relatório automático com gráficos e mapas |
| **4.3 Compartilhar resultados** | **Problema:** usuários podem ter dificuldade em distribuir resultados para equipe <br> **Recomendação:** permitir exportação em PDF ou compartilhamento direto |
