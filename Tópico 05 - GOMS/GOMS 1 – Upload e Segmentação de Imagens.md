# Modelo GOMS – Upload e Segmentação de Imagens 

---

# GOAL 1: acessar o sistema GeoSegment

- OP. 1.A.1: abrir navegador web  
- OP. 1.A.2: digitar endereço do sistema GeoSegment  
- OP. 1.A.3: pressionar **Enter** para acessar a página  
- OP. 1.A.4: visualizar tela inicial do sistema  

---

# GOAL 2: enviar imagem da área de interesse

## METHOD 2.A: iniciar upload da imagem
*(SEL. RULE: usuário possui imagem da área a ser analisada)*

- OP. 2.A.1: deslocar cursor para botão **Upload de imagem**  
- OP. 2.A.2: clicar no botão **Upload de imagem**  

---

# GOAL 2.B: selecionar imagem para upload

## METHOD 2.B.1: selecionar imagem manualmente
*(SEL. RULE: usuário prefere buscar arquivo nas pastas do computador)*

- OP. 2.B.1.1: navegar nas pastas do computador  
- OP. 2.B.1.2: localizar arquivo de imagem  
- OP. 2.B.1.3: selecionar arquivo (.tiff ou .png)  
- OP. 2.B.1.4: confirmar seleção do arquivo  

---

## METHOD 2.B.2: arrastar imagem para a interface
*(SEL. RULE: interface permite arrastar e soltar arquivos)*

- OP. 2.B.2.1: localizar imagem no computador  
- OP. 2.B.2.2: clicar e segurar arquivo  
- OP. 2.B.2.3: arrastar arquivo até área de upload  
- OP. 2.B.2.4: soltar arquivo na interface  

---

# GOAL 2.C: verificar envio da imagem

- OP. 2.C.1: observar barra de progresso do upload  
- OP. 2.C.2: verificar mensagem de confirmação do envio  

---

# GOAL 3: executar segmentação da imagem

- OP. 3.A.1: deslocar cursor para botão **Executar segmentação**  
- OP. 3.A.2: clicar no botão **Executar segmentação**  
- OP. 3.A.3: aguardar processamento do sistema  
- OP. 3.A.4: verificar geração do mapa segmentado  

---

# GOAL 4: visualizar resultados da segmentação

## METHOD 4.A: visualizar mapas gerados
*(SEL. RULE: usuário deseja analisar resultado visual)*

- OP. 4.A.1: observar mapa segmentado exibido  
- OP. 4.A.2: comparar mapa original e mapa segmentado lado a lado  

---

## METHOD 4.B: visualizar estatísticas por classe
*(SEL. RULE: usuário deseja analisar distribuição das classes)*

- OP. 4.B.1: localizar área de estatísticas  
- OP. 4.B.2: visualizar gráfico ou tabela de classes  
- OP. 4.B.3: interpretar porcentagens de cada classe  

---

## METHOD 4.C: visualizar histórico de segmentações
*(SEL. RULE: usuário deseja comparar mudanças ao longo do tempo)*

- OP. 4.C.1: selecionar opção **Histórico**  
- OP. 4.C.2: visualizar dados comparativos entre períodos  
- OP. 4.C.3: interpretar variações entre classes  

---
